# Known Limits

## Extraction quality

Some statement layouts and low-quality scans still need manual review. Bank format changes can also reduce accuracy until parser rules are updated.

## Sessions

Preview sessions are temporary. Save review edits before long pauses or before leaving the review page.

## Large files

Very large PDFs may hit page or file-size limits. Split large statements into smaller files when possible.

## When to contact support

Contact support if sign-in fails, uploads repeatedly fail for valid PDFs, the review screen cannot load, or billing usage changes unexpectedly.
