# Getting Started

## Sign in

Use the sign-in option provided to your account. The upload area appears after access is confirmed.

## Upload a statement

Select one PDF statement, choose the matching bank profile, and keep OCR enabled unless support asks you to retry without it.

## Preview first

The app opens a review screen before export. Check the transactions there before downloading the Excel file.

## What to keep nearby

- The bank name and statement month.
- The session ID shown on the review page.
- A screenshot if anything looks wrong.
