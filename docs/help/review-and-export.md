# Review and Export

## Review the table

Compare extracted rows against the PDF on the left side of the review screen. Rows marked for review need extra attention.

## Correct values

Edit incorrect transaction values in the table, then save changes before leaving the page.

## Export

Use the export action from the review screen after the rows look correct. The downloaded Excel file uses the reviewed transaction data.

## Good practice

Open the Excel file and check totals before using it in accounting or reporting.
