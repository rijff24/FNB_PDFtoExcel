# How to Use the App

Owner: Product + Engineering  
Last reviewed: 2026-04-15

## Quick usage guide

## Step 1: Sign in

1. Open the app home page.
2. Sign in with Google or Email/Password.
3. Wait for the upload section to appear.

## Step 2: Upload a statement

1. Click the file picker and select a PDF statement.
2. Keep OCR enabled unless support tells you to disable it.
3. Choose the correct bank profile when prompted.
4. Click **Preview and review**.

## Step 3: Review extracted transactions

1. Confirm rows look correct.
2. Edit incorrect fields directly in the table.
3. Use keyboard navigation if preferred.
4. Mark uncertain entries for manual follow-up.

## Step 4: Export to Excel

1. Click the export/download action from the review flow.
2. Open the downloaded file and verify totals.
3. Save the file into your normal accounting location.

## Step 5: Check billing and usage

1. Open **Billing and Usage**.
2. Review your current month totals and estimate.
3. Confirm OCR/non-OCR counts align with your activity.

## Recommended habits

- Review every statement before final export.
- Keep OCR enabled for scanned or complex layouts.
- Report issues with timestamp, bank name, and session details.

## If something goes wrong

- If no rows are extracted, retry with OCR enabled.
- If values look incorrect, correct them in review before export.
- If access or billing looks wrong, contact support using the Help topics.
