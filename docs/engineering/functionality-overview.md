# Functionality Overview

Owner: Product + Engineering  
Last reviewed: 2026-04-15

## Core features

## Authentication and access

- Sign in with Google or Email/Password.
- Account access is controlled by admin approval.

## Statement extraction

- Upload PDF bank statements from the home page.
- OCR toggle available (recommended on for most users).
- Parser support for configured/supported bank profiles.

## Review workspace

- Preview extracted transactions before export.
- Edit values directly in the review table.
- Keyboard-accessible controls for faster QA.

## Excel export

- Export reviewed transactions into an Excel file.
- Download generated files directly from the app.

## Billing visibility

- View monthly usage and pricing on the Billing page.
- Transparent pooled pricing breakdown for OCR and non-OCR usage.
- Live estimate during the month with month-end finalization.

## Support tooling

- Clear known limitations and expected behavior documentation.
- Support and escalation process for incidents and blockers.

## What this is not

- Not a full bookkeeping system.
- Not a guaranteed 100 percent parser for every PDF layout variation.
- Not a replacement for financial review controls.
